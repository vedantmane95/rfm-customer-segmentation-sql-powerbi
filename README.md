# 📊 RFM Customer Segmentation Dashboard – Power BI Project

## 📌 Project Overview

This project is an interactive **RFM (Recency, Frequency, Monetary) Customer Segmentation Dashboard built using SQL and Power BI**.
It analyzes customer transaction data to identify **high-value customers, at-risk customers, and overall purchasing behavior**.

The project follows a real-world workflow where data is processed in SQL (cloud-based) and visualized in Power BI for **data-driven business decisions**.

---

## 📁 Project Structure

* **dashboard.png** – Preview image of the dashboard
* **rfm_dashboard.pbix** – Power BI dashboard file
* **sales_data.csv** – Dataset used for analysis
* **rfm_queries.sql** – SQL queries for RFM calculation

---

## 📊 Dashboard Features

* **Customer Segmentation**

  * Champions
  * Loyal Customers
  * At Risk Customers
  * Lost Customers

* **RFM Metrics**

  * Recency (last purchase)
  * Frequency (number of orders)
  * Monetary (total spend)

* **Segment Distribution**

  * Customer count by segment
  * Revenue contribution by segment

* **Detailed Analysis**

  * Customer-level RFM scores
  * Behavioral insights
